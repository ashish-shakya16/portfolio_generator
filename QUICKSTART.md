## üìñ Step-by-Step Tutorial

### 1. Personal Information (2 min)
- Enter your full name
- Add your professional title
- Write or paste your bio
- Upload a profile photo
- Click "AI Improve" to enhance your bio

### 2. Skills (3 min)
- Paste comma-separated skills or add individually
- Use "Auto-Categorize" to organize
- Adjust skill levels (recommend 70-90%)
- Add 8-15 skills for best results

### 3. Education (2 min)
- Add your degree(s)
- Include institution name
- Add dates and GPA (optional)
- Save and continue

### 4. Experience (5 min)
- List your work history
- Add job responsibilities
- Include key achievements
- Use bullet points for clarity

### 5. Projects (5 min)
- Showcase 3-5 best projects
- Add GitHub and live demo links
- Use "AI Generate" for descriptions
- Include technologies used

### 6. Contact (1 min)
- Add email and phone
- Include LinkedIn, GitHub
- Add location (optional)

### 7. Choose Template (1 min)
- Browse 5 templates
- Preview each one
- Select your favorite

### 8. Customize (3 min)
- Pick colors or use presets
- Select font family
- Toggle sections on/off
- Preview changes

### 9. Export (1 min)
- Export as PDF for applications
- Export as HTML for deployment
- Deploy to hosting service

## üêõ Common Issues

**Q: Build fails with module errors**
A: Delete `node_modules` and `.next`, then run `npm install`

**Q: AI features not working**
A: Add OpenAI API key to `.env.local` file

**Q: Export not downloading**
A: Check browser popup blocker settings

**Q: Preview not showing**
A: Click "Show Preview" button in builder
